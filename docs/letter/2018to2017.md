# 2018-回复1年前的回信

发送时间: 2018年1月6日（星期六）下午3:45

## 回复1年前的回信

我终于有勇气、有能力去回复这封一年前的信了。

我想了二十多年的问题，终于找到答案了。

对比一年前的答案，我再调整一下吧。

第一，我不想成为像任何人那样的人，不管他是否有趣，就是不想成为像XX那样的人。我想成为自己了。

第二，我也不再过分的追求经济独立了。人人都说，女孩子要经济独立，才能自立自强，可，我问过自己是否真的想要自立自强的日子了吗？难道不是别人天天洗脑告诉我要赚钱养家、赚钱养活自己吗？没有绝对的独立，只有相对的。

第三、我已经在尽力去做了。

第四点：保留意见吧，因为已经不重要了。

我的梦想历程：

- 2011年：文员
- 2012年：黑客
- 2013年：软件四级
- 2014年：软件工程师
- 2015年：同上
- 2016年：全栈工程师
- 2017年：成为自己


我知道，并不是今天的答案，就是最后的答案了，每一年，经历得不同，见到世界的宽度不同，格局就不同了。学生时代，我的梦想只能建立在摸不到的虚拟物品上，也许黑客、全栈是我喜欢的，但是绝不会100%是我喜欢的，那其中，夹杂着对生活的妥协，对未知生活的害怕和向往，对父母尽孝的迫不及待……

今天，终于找到了一个100%是自己喜欢的梦想，终于发现，自己才是最重要的。先爱自己，才是爱别人的基础。

记得很清楚，半年前收到这封邮件时，是特别的迷茫，日子过得不上不下，不好也不差。我实在是形容不出那时的心情，十分自责自己怎么不小心把那么努力的自己给丢了。

那时候会有那样的心情，现在也是十分理解。因为我根本就不知道自己想要什么了。

以前一直心心念念追求的、全是和IT相关的，从事这个职业之后，硬生生的把爱好变成了饭碗，那种感觉不自由，在自己能力不足的情况下，不自由。所以，十分迷茫。

现在觉得，如果十分努力的自己并不是我喜欢的，不要也罢，我不要十分，七分就够了，要那么拼命干嘛呢？伤身体不说，也会让人慢慢变得消积的。

可能没人能理解，一年后找到梦想后为什么会这么高兴，哈哈，还是一个看起来不像梦想的梦想。。。

只有我自己知道，这个过程多么艰辛。

一个人能以“爱自己”为梦想，总归是经历了点什么的。
