# 2016-给一年后自己的一封信

发送时间: 2016年12月25日（星期日）下午1:40

## 给一年后自己的一封信

好土，好俗的方式啊，可是这样又让生命多了一丝趣味不是吗？

当收到这封邮件的时候，一定是很惊喜的吧。

现在是2015年12月25日，对我而言，没有所谓的圣诞，怪诞还差不多。不过没关系啦，这不是我想跟你表达的。嗯，也许你知道我要说什么呢， 嘿嘿嘿。

今天天气很好，不知道一年后的我收到这封信的时候，天气是否也一样挺好的呢？

早就满了十九岁，再过几天，就要虚岁二十了。想不到，竟然这么快就长大了呢，还来不及回首往事，就蹭蹭蹭的长大了，当然，并没有长高。

我不想开门见山的就说希望你能坚持自己，我希望能看到一些温暖，所以我努力把自己现在的想法记录下来，我不希望，我的未来，会变得那么索然无味、机械死板，只知道去认同真理。

反正这是 写给自己看的信，无所谓格式，无所谓语句通顺啦！

一年后的我，应该满二十虚岁二十一了吧！记得小时候我喜欢把人生5年5年的分开。那么这一年应该是你开始的第一年了。岁月无情，那么，你还记得当初的梦想吗？还记得那个弱小的自己吗？是在一步步变强大，还是在退缩呢？我没有资格说你什么，毕竟我没有到一年后，我不知道一年后的自己会发生什么。可是如果我不说你的话没有人会说你，你不会醒，你会沉下去，你会退进平庸里再也起不来。我就是你，没有人比我更了解你了。（话说不知道别人会不会以为我有病，自己和自己聊天就算了，还自己和自己发邮件，哈哈。）

别放弃，好不容易来到这个世界上，可以被消灭，但别被打败。

你还喜欢看书吗？今年的十一月，我可是一口气看完了七本书哦。你呢？有没有被生活压低了头然后就再也抱振作不起来了？

加油啊，别放弃啊，别说什么我不知道你现在的痛苦，我怎么可能不知道，我就是你啊！

有多少次，为程序犯难，愁得我牙痒，恨不得剥皮抽筋，今程序之魂，可是还是没用，要一步一步慢慢来。更何况，程序教给我的，不全是代码，还有很多，生活中的真理。

你还在坚持写代码吗？你还在坚持早起吗？你还在坚持读书吗？

没坚持了也没关系，只要你一直在坚持向上。

以上。
