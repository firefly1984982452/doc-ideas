# 2017-向五年后的自己提问

发送时间: 2017年7月3日(星期一) 晚上9:40

## 向五年后的自己提问

今天，收到了很多朋友的QQ祝福，生日快乐。但实际上，这个日期是我捏造的，是我申请QQ的那一天。然后我“哦”了一声，原来，五年过去了。

五年前的今天，高中提前班，再过一个月，也就是遇到梦的整整五年了。五年前，我没给自己提问五年后也就是今天的我会是什么样子，但估摸着，那时候的我想要的人生，就是三点一线的生活，普通平凡的日子，简单的文员。因为那时候所接触到的世界最大限度就是那个样子了，那已经是个很了不起的梦想了。

今天呢？常想问自己五年后，但总是不敢回答，怕，怕不是心底最终的答案，怕梦太远，触不可及；怕梦太近，辜负所有人的厚爱。

正好一年，恰逢《精进》，那今天就解决这个问题吧。

- 1.成为一个像毛帅那样有趣的人
- 2.经济独立
- 3.思维、思想、思考上升层次
- 4.成为一个有趣的程序媛

每一个阶段的梦想，很大程度取决于该阶段的状态，包括心情、阅读、环境、所了解的世界。比如最近在阅读《精进》和《暗时间》，所以对思维有很大要求。

此为手稿，将录入，发邮箱，半年后收一次，五年后收一次。

我的梦想经历过怎样的变化呢？

- 2011年：文员
- 2012年：黑客
- 2013年：软件四级
- 2014年：软件工程师
- 2015年：同上
- 2016年：全栈工程师
